# ATM Simulation System

A simple ATM Simulation project built in Python.  
This project simulates basic ATM operations like checking balance, withdrawing money, depositing money, and PIN verification.

---

## Features
- User PIN authentication  
- Check account balance  
- Deposit money  
- Withdraw money  
- Exit system
